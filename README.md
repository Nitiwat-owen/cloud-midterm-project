Getting Start 
1. clone the repository
2. download dependencies by `go mod download`
3. create `.env` for connect to the database
    - `DB_USERNAME`
    - `DB_PASSWORD`
    - `DB_NAME`
    - `DB_HOST`
 4. run server by `go run main.go`
